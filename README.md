# Attendance App

## Purpose
Track attendance efficiently for classes, events, and workplaces.  
The goal is to make attendance management simple, fast, and insightful.

## Features (Planned)
- User authentication
- Attendance tracking and analytics
- Session-based updates for space efficiency
- Web app with future mobile integration

## Tech Stack
- **Backend:** Flask (Python)
- **Frontend:** HTML/CSS/JS (later React)
- **Database:** SQLite (initially)
- **Version Control:** Git

## Component Overview
- **Flask:** Handles backend logic and routes.
- **Virtual Environment:** Keeps dependencies isolated.
- **Git:** Tracks version history and collaboration.
- **SQLite:** Stores attendance data locally for testing.
